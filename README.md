# Paradise Nursery

## Project Description
Paradise Nursery is a React-based e-commerce web application designed for a houseplant nursery. This project provides users with a seamless shopping experience, featuring an attractive landing page, a detailed product listing page, and a fully functional shopping cart managed by Redux.

## Features
* **Landing Page:** A welcoming interface featuring the company name and a "Get Started" button.
* **Product Listing:** Displays various houseplants organized into distinct categories.
* **Shopping Cart:** Users can add plants to their cart, increment/decrement quantities, and view the total cost. Built with Redux for state management.
